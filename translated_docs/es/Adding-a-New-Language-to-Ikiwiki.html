Cómo añadir un nuevo idioma a Ikiwiki

A continuación se muestran instrucciones para añadir un nuevo idioma al sitio web addons.nvda-project.org, que usa Ikiwiki.

  1. Edita el archivo ikiwiki.setup en el repositorio ikiwiki-ctl. La URL del repositorio Git es: git@bitbucket.org:nvdaaddonteam/ikiwiki-ctl
  2. Añade una nueva entrada bajo la opción po_slave_languages con el nuevo idioma, siguiendo el formato de las otras entradas.
  3. Guarda, haz commit y luego push.
  4. Inicia sesión en exbi y cambia a la cuenta de usuario nvdal10n.
  5. cd ~/ikiwiki/ctl
  6. git pull
  7. ikiwiki --setup ikiwiki.setup --verbose
